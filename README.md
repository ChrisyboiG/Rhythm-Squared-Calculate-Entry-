# Rhythm-Squared-Calculate-Entry-
This is my project for the Calculate thingy on Hackclub, my first python and pygame project

Run "pip install pygame" and that should be good.

it uses song.nt files as the format for the notes that are dropped.
It wil play song.mp3 while playing.

Use the UIOP keys to hit the notes in time, the correct notes will be disguised in the fakes, solve the equation to find the right number.
The red line changes the current equation every 15 seconds.

Play the bonus number catcher minigame too!

Beware the game can be really difficult!
